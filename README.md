# MapKittest333
При запуске приложения(карты), загружается карта. Затем двигаем карту в право немного до 2 - х указателей. При нажатии на них всплывает вью, по которой тапаешь и можно работать с геопозтцией. Элементарный давний проект, но может кому пригодится.
